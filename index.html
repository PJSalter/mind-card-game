Game Instructions:

  • The aim of the game is to win a pair of the same sea life creatures.

  • You as a user must be tacticle with memory in order to guess them correctly.

  • click on your mouse once for the first card then once for the second.

  • choosing the wrong pair will cause them to shuffle back.

  • Once the user gains the correct pair then the card places will change image to a star fish at the bottom of the ocean.

  • Choosing all the correct pairs wins the game completely. There are 6 pairs in total.

  • As the congratulations message pops up once finishing, click the restart button to start playing again.
